Color

Color is a Firefox extension that allows users to create and share custom themes for their browser. I was contracted by Mozilla to work on the Color project to prepare to move it out of the Test Pilot program. I collaborated with Add-ons team members around the world to rfix bugs and update the code. Color is written in HTML, CSS, JavaScript, and React. Create your own theme or see the code here.

Fun eCards

Fun eCards takes user input and creates madlibs-style greeting cards. I used Python and Django to build the project and utilized CSS media queries to format the output for optimal results when printing or saving to PDF. I deployed the app through Heroku, so you can make your own Fun eCards! Or take a look at the code here.

Tri-City Collective site

Tri-City Collective is group of writers, artists, and teaachers that produces creative works and affects cultural change through social justice and community service. I built a site that combined their unique vision with an intuitive user interface using WordPress themes, widgets, and custom CSS styling. I also designed a simple branding package that included logos for their website and social media presence. Click here to take a look at the site.
